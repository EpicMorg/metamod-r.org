    Metamod-r...

    ...is based on the original version of Metamod written by Will Day for Half-Life 1 with some improvements from Jussi Kivilinna (Developer of Metamod-p ). This product contains a large number of performance optimizations and more pure code. The core was written using JIT compiler.

    Metamod-r is incompatible with original HLDS . It's necessary to have installed ReHLDS (API 3.1+) . There is no guarantee that the product will work in a different environment.
